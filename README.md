# 🏥 Clinic Management System

## 📌 Project Overview
This project is a **Clinic Management System** designed for beginner students (1st year) to learn how to work with databases using PostgreSQL.

The main goal is to understand:
- Database design
- SQL queries
- Relationships between tables
- Basic backend integration

---

## 🎯 Objectives
This project helps students learn:

- CRUD Operations (Create, Read, Update, Delete)
- Table Relationships (Foreign Keys)
- JOIN Queries
- Aggregation Functions (SUM, COUNT)
- Database Normalization basics

---

## 🧩 System Features

### 👥 Patients Management
- Add new patients
- View patients list
- Delete patients
- Update patient information

---

### 👨‍⚕️ Doctors Management
- Store doctor details
- Assign doctors to appointments

---

### 📅 Appointments System
- Schedule appointments
- Link patients with doctors
- Manage appointment status (confirmed, canceled, pending)

---

### 💰 Invoices System
- Create invoices for patients
- Calculate total amounts
- Track billing history

---

### 💊 Prescriptions
- Doctors create prescriptions
- Each prescription contains multiple medications

---

### 🛏️ Rooms & Admissions
- Manage hospital rooms (normal, VIP, emergency)
- Admit and discharge patients

---

### 🧪 Lab Tests
- Store available lab tests
- Record patient test results

---

### 📞 Reception System
- Manage patient queue
- Track waiting and service status

---
